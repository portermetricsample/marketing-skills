# Google Ads — Search Terms Page

The Search terms page as a live component: keyword cards with per-term criterion tags that surface only when a term needs attention.

- **Files & usage:** [`README.md`](README.md)
- **Live preview:** [`demo.html`](demo.html)
- **Sample data:** `example.data.js` (fictional Acme Insurance)

Appearance is owned by the public **porter-design** repo (`portermetricsample/porter-design`); the component ships no CSS. Data comes from the Porter Metrics `google-ads` connector.
