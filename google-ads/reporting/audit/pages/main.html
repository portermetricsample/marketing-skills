Executive summary

$48K spent this quarter — but the account is measuring the wrong thing, so we don't actually know if it's working.

Google Ads recorded 214 leads in Q1. The real number is 28 — the other 186 were people who just visited the homepage. Do first: Fix what Google Ads counts as a lead (§01) — everything else is noise until that number is right. Then cut the searches that are wasting money (§06). Then reset the cost-per-lead targets to match reality (§02).

§01

Are we counting leads correctly? Conversion Tracking

Broken HIGH

The account thinks it got 214 leads this quarter. The real number is 28. The other 186 "leads" are people who simply visited the homepage — not a lead at all. This happens because Page View is set as the main action Google Ads optimizes toward, instead of the contact form. Every cost-per-lead number in this account is calculated on the wrong base.

What we checked State Evidence Recommendation
Google Ads is optimizing toward real leads Is the contact form the main goal — not just a page visit? Broken Primary conversion: "Page View — Homepage" (186 of 214 conversions) Where: Google Ads → Tools → Conversions.
What: Set "Page View" to Secondary. Set "Book a consultation" to Primary.
Why: Homepage visits inflate the lead count 7.6× — bids chase traffic, not real inquiries.
Each lead has a dollar value assigned Does Google Ads know what a lead is worth? Broken All conversions = $0 value. tROAS bidding not available. Where: Google Ads → Tools → Conversions → each action.
What: Assign a value (e.g. $750 average deal value).
Why: Without a value, Google can't optimize for revenue — only volume.
Closed deals are sent back to Google Ads Do we tell Google which clicks actually became clients? Review No CRM import found. Google only sees form fills, not won deals. Where: Google Ads → Tools → Conversions → + New → Import.
What: Import closed-won deals from CRM (Salesforce / HubSpot).
Why: Tells Google which leads actually converted — bids improve over time.
Old Analytics goals are turned off Are there legacy tracking goals still counting from the old setup? Pass No legacy Universal Analytics goals active. No action needed.
Fix
Change the main goal from "page visit" to "form fill" this week. In Google Ads → Tools → Conversions: demote "Page View — Homepage" to Secondary, promote "Book a consultation" to Primary. One change — corrects every efficiency number in the account. Until it's done, all the data below is approximate.
Note
All cost-per-lead figures in this audit use the 28 real form fills, not the 214 recorded by the account. Section numbers have been corrected accordingly.

Because Google thinks the cost per lead is $225 (based on page views), it's bidding as if the account is performing well — §02 shows just how far off the actual bid targets are.
