# Google Ads — Financial Overview (executive)

> A **single-screen** executive view for someone who is NOT in marketing
> (manager, finance, owner). Only the metrics that matter to whoever puts up the money,
> compared vs the previous period.

- **Who is it for?** Manager / leadership / finance. Does not talk about CTR or keywords.
- **When is it used?** Periodic summary (typically monthly); a 30-second read.
- **What decision does it help make?** Is Google Ads delivering a return and are we doing
  better or worse than last period? Yes/no, without getting into operational detail.

## What it shows
- **5 big KPIs (totals):** Conversions · Conversion value · Spend ·
  Cost per conversion (CPA) · ROAS — each vs the previous period.
- **1 breakdown table:** conversions broken out by **Name** and **Type**, with
  Conversions, Value, **Value per conversion**, **% of value** and a
  **primary/secondary** flag (no spend/CPA per row — see `framework.md` for why).
