# Strategy Skills

Three skills that structure marketing thinking from strategy to copy. Each covers one layer of the marketing hierarchy and feeds the next — so every asset traces back to explicit strategic decisions instead of improvisation. Company- and industry-agnostic.

## The hierarchy

| Layer | Question | Skill |
|---|---|---|
| 1. Strategy (Segmentation + Targeting) | Who is this for, and why them? | `stp-marketing-strategy` |
| 2. Positioning | What do we want to own in their mind vs their alternatives? | `stp-marketing-strategy` |
| 3. Narrative | What's the story that makes the position urgent? | `positioning-messaging` |
| 4. Messaging | What arguments do we make? | `positioning-messaging` |
| 5. Asset / Copy | What are the actual words? | `landing-page-copy` |

Each layer is derived from the one above, never invented at its own level. Diagnosis runs upward: a weak landing page is usually a messaging problem, and weak messaging is usually a positioning problem.

## The skills

### 1. `stp-marketing-strategy`
The STP framework: segmentation (JTBD-first, MASDA-validated), targeting (scored decision + ICP with disqualifiers), positioning (Dunford's five components, positioning statement, four stress tests). Use it to build a strategy or to plan/critique any marketing work against one.

### 2. `positioning-messaging`
Turns positioning into words: a strategic narrative (Andy Raskin's five moves — change in the world, winners and losers, promised land, magic gifts, evidence) and a messaging doc (point of view, 2–3 value themes with claims + proof, objections, customer vocabulary). Every claim must pass the competitor name-swap test and carry proof.

### 3. `landing-page-copy`
Structures and writes landing pages: match structure to the traffic's awareness stage (Schwartz: unaware → most aware), select the section layout accordingly (narrative long-form / canonical SaaS / short direct), fill every section from the messaging, then hold the copy to a quality bar (required components, adjective test, brevity rules, pre-publish checklist). Includes a critique mode that traces each section back to messaging and flags orphan claims.

## Typical flow

```
"Define our strategy"            → stp-marketing-strategy
"What should we say about it?"   → positioning-messaging
"Write the landing page"         → landing-page-copy
"Review this page"               → landing-page-copy (+ upward diagnosis)
```

Skills also work standalone: each checks for the inputs it needs and asks for (or reconstructs) what's missing.
